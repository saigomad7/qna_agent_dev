## LangGraph 핵심 기술 구성 요약

| 컴포넌트 | 적용 위치 | 역할 |
|----------|----------|------|
| `StateGraph` + `TypedDict` | A1, A2 모두 | 에이전트 상태 정의 및 실행 흐름 관리 |
| `ToolNode` | A1 데이터 수집, A2 검색 | 외부 API / 벡터 DB 호출 도구 등록 |
| `Conditional Edge` | A1 이상 감지 라우팅 | 감지 결과에 따른 분기 처리 |
| `interrupt()` | A1 알림, A2 사용자 확인 | Human-in-the-Loop 구현 |
| `Send` API | A2 병렬 검색 | 서브 질문별 병렬 Retrieval 실행 |
| `Subgraph` | A1/A2 모듈화 | 기능 단위 그래프 분리 및 재사용 |
| `Supervisor` | 통합 오케스트레이터 | W21~W22에서 A1/A2 최상위 조율 |
| `MemorySaver` / `AsyncPostgresSaver` | 전체 | 개발/운영 Checkpointer 분리 |
| `stream_mode` | A2 인터랙티브 응답 | 실시간 스트리밍 응답 제공 |
| LangSmith | 전체 | 트레이싱, 디버깅, 비용 추적 |
